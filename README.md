# Techbang::Capistrano

Capistrano recipes for Techbang.

## Installation

Add this line to your application's `Gemfile`:

```ruby
group :development do
  gem 'techbang-capistrano', :require => false
end
```

## Usage

In `config/deploy.rb`:

```ruby
require 'techbang/capistrano'
```
